# podcast_accesibility
Web con accesibilidad cognitiva hecha con React que esta adaptada para los siguientes problemas:

• Discapacidad visual (ceguera, baja visión)
• Discapacidad auditiva
• Limitaciones motoras (dificultad para usar ratón)
• Dificultades cognitivas
• Usuarios que navegan con teclado
• Usuarios que utilizan lectores de pantalla
